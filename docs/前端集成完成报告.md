## 🔧 部署和使用

### 快速启动
```bash
# 1. 运行数据库迁移
python manage.py migrate

# 2. 创建演示数据
python manage.py create_demo_data

# 3. 启动开发服务器
python manage.py runserver

# 4. 访问系统
http://127.0.0.1:8000/
```
